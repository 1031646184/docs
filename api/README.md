## Baobaozhuan Api 设计文档

### 如何使用

1. 打开 [Swagger Editor](http://editor.swagger.io/)

2. 将该 api.yaml 文件内容复制到编辑器中

3. 右侧将显示可交互的动态api文档，可查看并测试
